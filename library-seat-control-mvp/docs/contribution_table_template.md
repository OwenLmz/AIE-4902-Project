## 填写注意事项

- 只填写真实完成的工作。
- 每项贡献尽量对应一个文件、截图、测试结果或文档。
- 不要把 AI 生成内容直接写成某位成员的独立贡献。
- 如多人共同完成同一模块，应标注“共同完成”并说明每人的具体部分。
- 与真实数据、摄像头、闸机、模型权重相关的证据提交前必须做合规确认。
