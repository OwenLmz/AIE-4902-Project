# AI / Tool-use Disclosure 草稿

日期：2026-06-26

本项目在课程 MVP 开发与中期材料整理过程中使用了 AI 工具辅助。团队对最终提交内容的正确性、安全性和原创性负责。

## 1. 使用的 AI 工具

- ChatGPT
- Codex

## 2. 使用目的

AI 工具主要用于：

- 项目讨论和需求澄清；
- PRD 结构整理；
- 数据申请文档整理；
- 前端 UI 审计和重构建议；
- Python / Streamlit MVP 代码生成；
- 测试用例生成；
- 文档整理；
- 中期报告大纲和草稿整理；
- evidence 清单和检查清单整理。

## 3. AI 输出如何被验证

团队通过以下方式验证 AI 辅助产物：

- 人工审查需求边界和功能范围；
- 人工确认不虚构真实数据、预测结果和成员贡献；
- 单元测试：`python3 -m unittest discover`；
- 编译检查：`python3 -m py_compile ...`；
- Streamlit 健康检查：`curl -I http://localhost:8501`；
- 禁止项搜索，检查是否出现随机业务数据、假处理状态、管理员操作按钮、个人身份信息等；
- 功能边界审查，确认当前阶段不接入真实 API、不展示原始图像、不实现管理员处理闭环。

## 4. 已知限制

- AI 可能生成超出 MVP 范围的不必要功能。
- AI 可能虚构不存在的数据字段。
- AI 可能使用不合适的展示形式，例如给学生端展示管理员检测细节。
- AI 可能把 mock 数据描述得过于接近真实系统。
- AI 可能忽略隐私、数据合规或课程项目边界。

团队通过 UI 审计、重构计划、测试、编译检查、健康检查和人工文档复核降低上述风险。

## 5. 责任声明

团队对最终提交内容的正确性、安全性、原创性和合规性负责。AI 输出仅作为辅助材料，不能替代团队成员的理解、审查、验证和最终判断。

最终报告中不应声称 AI 工具完成了未经团队验证的事实，也不应把 AI 生成内容直接等同于某位成员的独立贡献。
