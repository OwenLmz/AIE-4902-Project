# AIE 4902 Midterm Report LaTeX Project

This folder is a collaborative LaTeX project for the team midterm report.

## Files

- `main.tex`: main report entry point.
- `sections/`: required report sections from the midterm requirements.
- `appendices/`: contribution and AI-use appendix.
- `figures/`: put screenshots, architecture diagrams, or evidence images here.

## How teammates should edit

1. Data structure teammate: fill data structure subsections and contribution rows.
2. Frontend teammate: fill frontend subsections and contribution rows.
3. Image algorithm section is already drafted and can be revised if needed.
4. Add screenshots to `figures/` and reference them from the relevant section.
5. Keep the final report around 8-12 pages excluding appendices.

## How to share

You can upload this whole folder to Overleaf, or commit it to GitHub and share the repository link with teammates.

## Compile

Compile `main.tex`. If using Overleaf, choose pdfLaTeX first. The template avoids Chinese text in the LaTeX source to keep compilation simple.
